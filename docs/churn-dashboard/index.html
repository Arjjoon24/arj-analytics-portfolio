Customer Churn Analysis Dashboard

Overview

This dashboard presents an analysis of customer churn patterns for a telecom service provider. Insights were derived using SQL queries and visualized in a Tableau-style format.

Key Insights

  • Churn rates are highest among Basic plan customers.
  • Customers with high complaint levels are significantly more likely to churn.
  • Retained customers tend to use more data monthly compared to churned customers.
